## target

- 符合标准的JSON解析器和生成器

- 手写的递归下降解析器和生成器 (recursive descent)

- 使用标准C语言(c89)

- 跨平台支持

- 仅支持UTF-8 Json文本

- 仅支持double存储和json number类型
